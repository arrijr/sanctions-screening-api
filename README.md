# Sanctions & PEP Screening API

Screen individuals and companies against 100+ global sanctions lists including
OFAC SDN, EU Consolidated Sanctions, UN Security Council, Interpol, and PEP databases.

**Available on [RapidAPI](https://rapidapi.com/arrijr/api/sanctions-screening-api)**

## Endpoints

- `POST /api/v1/screen` — Screen a single person or company
- `POST /api/v1/screen/batch` — Screen up to 10 entities (Pro+)
- `GET /api/v1/datasets` — List available screening datasets
- `GET /api/v1/health` — Health check
